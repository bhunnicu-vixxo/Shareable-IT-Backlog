# Keyboard Shortcuts

The Shareable IT Backlog application is fully navigable using only a keyboard, following WCAG 2.1 Level A guidelines.

## General Navigation

| Shortcut | Action |
|---|---|
| `/` | Focus the search bar from anywhere on the page (does not trigger when typing in an input or when a modal is open) |
| `Tab` | Move focus to the next interactive element |
| `Shift + Tab` | Move focus to the previous interactive element |
| `Enter` | Activate the focused element (buttons, links, cards) |
| `Space` | Activate the focused element (buttons, toggles) |
| `Escape` | Close an open modal; clear and blur the search input |

## Skip Navigation

| Shortcut | Action |
|---|---|
| `Tab` (first press on page) | Reveals the "Skip to main content" link |
| `Enter` (on skip link) | Jumps focus past the header and filters to the main content area |

## Backlog List

| Shortcut | Action |
|---|---|
| `Arrow Down` | Move focus to the next backlog item in the list |
| `Arrow Up` | Move focus to the previous backlog item in the list |
| `Home` | Move focus to the first backlog item |
| `End` | Move focus to the last backlog item |
| `Enter` / `Space` | Open the detail modal for the focused backlog item |

## Modals (Item Detail)

| Shortcut | Action |
|---|---|
| `Tab` | Cycle focus between interactive elements within the modal (focus is trapped) |
| `Shift + Tab` | Cycle focus backwards within the modal |
| `Escape` | Close the modal and return focus to the triggering element |

## Filter Controls

| Shortcut | Action |
|---|---|
| `Enter` / `Space` | Open a dropdown (Business Unit filter, Sort control) |
| `Arrow Down` / `Arrow Up` | Navigate options within an open dropdown |
| `Enter` | Select the highlighted dropdown option |
| `Escape` | Close a dropdown without selecting |
| `Enter` / `Space` | Toggle the "Show only new items" button |

## Admin Page

| Shortcut | Action |
|---|---|
| `Tab` | Move focus through controls on the Admin page (tabs and actions) |
| `Arrow Left` / `Arrow Right` | Switch between tabs |
| `Enter` / `Space` | Activate admin action buttons (Approve, Disable, Enable, Sync Now) |

## Focus Indicators

All interactive elements display a visible green outline (Vixxo Green, 2px solid) when focused via keyboard. This ensures keyboard users can always identify which element currently has focus.
